# may-mobile

MAY platformu Flutter mobil uygulamasi.

## Teknolojiler

- **Flutter** (Dart SDK ^3.6.0)
- **Riverpod** — State management
- **GoRouter** — Routing
- **Dio** — HTTP client
- **JWT** — Authentication

## Ozellikler

- Login / Logout (JWT)
- Dashboard
- Kullanici yonetimi (CRUD)
- Rol yonetimi (CRUD)
- Yetki yonetimi (CRUD)
- Kullanici-Rol iliskileri
- Rol-Yetki iliskileri
- Ayarlar

## Kullanim

```bash
# Bagimliliklari yukle
flutter pub get

# Calistir
flutter run

# Build (APK)
flutter build apk
```
